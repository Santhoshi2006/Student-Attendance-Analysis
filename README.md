📊 Student Attendance Analysis (Data Mining)
📌 Project Overview

This project focuses on analyzing student attendance data using data mining techniques to discover patterns, trends, and insights that can help improve academic performance and student engagement.

The analysis is performed using the WEKA tool, applying preprocessing and transformation techniques to prepare the dataset for meaningful insights.

🎯 Objectives
Analyze student attendance data
Identify patterns across different classes and sections
Apply data preprocessing techniques
Generate insights to improve academic performance
🛠️ Tools & Technologies
WEKA (Waikato Environment for Knowledge Analysis)
Data Mining Techniques
Dataset (Student Attendance Data)
🔄 Data Preprocessing Steps

The dataset was processed using the following steps:

Data Cleaning
Removed inconsistencies and missing values
Data Transformation
Normalization of values
Data Reduction
Removed unnecessary attributes
Discretization
Converted numerical data into ranges (e.g., 0.8–0.9, 0.5–0.6)
📂 Dataset Description

The dataset includes the following attributes:

Name – Student name
Class – Academic class
Section – Section of the student
Total Marks – Performance indicator (discretized ranges)
Attendance – Attendance percentage
📊 Key Insights
Students with higher attendance tend to have better performance
Attendance patterns vary across sections
Low attendance groups can be identified for improvement
📈 Results

The project successfully demonstrates how data mining techniques can:

Extract useful knowledge from raw data
Support academic decision-making
Improve student performance tracking
